# Nurse Guide

## Login and Dashboard
- Login with Nurse role.
- You are redirected to care operations (OPD/IPD support) dashboard.

## Core Modules
- OPD/IPD, patient vitals/care updates, medicines dispensing support, appointment visibility.

## Daily Tasks
- Update patient care details and nursing notes where permitted.
- Coordinate admitted patient bed/ward updates.
- Support medication administration workflows.

## Exports
- Export available lists (admissions/assignments) to CSV/PDF when enabled.

## AI Assistant
- Use bottom-right AI button for quick workflow/navigation guidance.
