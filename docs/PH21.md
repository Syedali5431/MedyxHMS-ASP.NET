# PH21 - Theme Preset Selector for Public Style Studio

## Objective
Add quick-start visual presets to Public Site Settings so Admin and SuperAdmin can apply curated themes instantly, while preserving full manual control.

## Scope Delivered
- Added `ThemePreset` selector with options:
  - Custom
  - ClinicalBlue
  - WarmCare
  - HighContrast
- Selecting a preset auto-fills style fields:
  - primary color
  - accent color
  - surface color
  - heading style
  - button style
- Manual style edits automatically switch preset to `Custom` when values diverge.

## Data and Persistence
- Added persisted setting key `PublicSiteThemePreset`.
- Included default seed (`Custom`) for existing environments.

## Access and Boundaries
- Feature remains under existing Public Site Settings authorization (`Admin,SuperAdmin`).
- Dashboard/module governance remains unchanged (SuperAdmin-only control for section/module enablement).

## Validation
- Build and tests pass after adding preset field, save/load behavior, and client-side interactions.
