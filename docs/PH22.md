# PH22 - Visual Preset Swatch Cards in Style Studio

## Objective
Improve preset discoverability and speed by adding clickable visual preset cards in Public Site Settings.

## Scope Delivered
- Added clickable swatch cards for:
  - ClinicalBlue
  - WarmCare
  - HighContrast
  - Custom
- Card click behavior:
  - sets preset dropdown value
  - applies matching preset values (except Custom)
  - refreshes desktop/mobile live previews
- Added active-state highlight for selected preset card.
- Added bidirectional sync:
  - dropdown changes update active card
  - manual style edits that diverge from preset switch to `Custom` and update card state

## UX Notes
- Presets are now discoverable visually before applying.
- Users can still fine-tune all fields after applying a preset.

## Access and Boundaries
- Available under existing Public Site Settings role guard (Admin, SuperAdmin).
- No change to module/dashboard governance boundaries.

## Validation
- Build and tests pass after card UI and script enhancements.
