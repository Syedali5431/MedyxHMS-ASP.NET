# PH18 - Public Style Studio Extension (Color, Heading, Button Controls)

## Objective
Provide Admin and SuperAdmin with richer, low-code visual controls for the public website while keeping dashboard governance restrictions unchanged.

## Scope Delivered
- Added Style Studio controls in Public Site Settings for:
  - Primary color
  - Accent color
  - Surface/background color
  - Heading style (normal/uppercase/capitalize)
  - Button shape (rounded/pill/square)
- Applied style settings globally in the public layout.
- Seeded defaults for new style settings so existing environments remain stable.

## Access Model
- Public style editing remains under `PublicSiteAdminController` with `Admin,SuperAdmin` role authorization.
- Dashboard section/module governance remains SuperAdmin-only in module management endpoints.

## Validation
- Compiles with existing architecture and settings model.
- Uses fallback values and color sanitization at render time for safe display.

## Notes
- This release intentionally avoids dashboard style changes; all customization targets the public-facing site only.
