# Doctor Guide

## Login and Dashboard
- Login with Doctor role.
- You are redirected to OPD/clinical workflow screens.

## Core Modules
- OPD, IPD (assigned), Appointments, Patient records, Lab/Radiology requests, Prescription.

## Clinical Workflow
- Review appointments and patient history.
- Add/edit clinical notes and diagnosis details where permitted.
- Request lab and radiology tests.
- Issue prescriptions.

## Patient Tracking
- Review patient payment/report status as needed for treatment continuity.
- Monitor admitted patient bed and ward details in IPD context.

## Export and Reports
- Export visit-related tables as CSV/PDF where enabled.
- Access report outputs based on assigned permissions.

## AI Assistant
- Use floating AI button to get help with module navigation and process guidance.

## Account Notes
- Doctor accounts created by Admin/SuperAdmin now require unique `user_name` values.
- New doctor accounts created via signup remain inactive until approved.
